# OCR-Korean-Helper-Application-using-CNN
[한국산업기술대학교 종합설계] 외국인 여행객을 위한 OCR 한글 도우미 어플리케이션

## Introduction
 간판 등의 이미지를 촬영하여 글자를 인식, 영어로 해석하주는 어플리케이션 개발합니다. 이를 통해 영어를 사용하는 외국인여행객을 위한 번역 정보를 제공합니다.
